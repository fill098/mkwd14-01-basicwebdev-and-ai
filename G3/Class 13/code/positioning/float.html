One

This paragraph shows how text wraps around an image that is floated to the left. The image stays on the left side, and the text flows nicely around it. Floats were originally created for this type of layout (like images in articles).

When there is no more text, the content below will continue normally. Floats can affect following elements unless we clear them.

This text appears *below* the floated image because we used a clear element.

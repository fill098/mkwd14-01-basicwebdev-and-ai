# HTML Practice Exercise: Beginner Level

This exercise will help you practice basic HTML skills without using
advanced CSS.\
You will use **headings**, **links**, **navigation**, **lists**,
**tables**, **images**, and **external links**.

## ðŸ§© Requirements

### 1. Page Title & Headings

-   Add a title in the `<title>` tag.
-   Use headings: `<h1>`, `<h2>`, and `<h3>`.

### 2. Navigation with Links

Create a navigation menu at the top using a list and links. For
example: - Home - About - Gallery - Contact

Use:

``` html
<nav>
  <ul>
    <li><a href="#home">Home</a></li>
    <li><a href="#about">About</a></li>
    <li><a href="#gallery">Gallery</a></li>
    <li><a href="#contact">Contact</a></li>
  </ul>
</nav>
```

### 3. Colors and Fonts (external css file)

-   Change the color of some text.
-   Change a font.
-   Use the box model

### 4. Insert an Image

Use any image stored in a folder, e.g.:

``` html
<img src="images/myphoto.jpg" alt="Description of image" width="300">
```

### 5. Create a List

Add a section that includes: - An **ordered list** - An **unordered
list**

### 6. Build a Simple Table

Include a table with: - A table header (`<thead>`) - A body
(`<tbody>`) - 2--3 rows of data\
Example: favorite movies, daily tasks, sports, etc.

### 7. External Link

Add a link to an external website, such as:

``` html
<a href="https://www.wikipedia.org" target="_blank">Visit Wikipedia</a>
```

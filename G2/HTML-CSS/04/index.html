Git Basic Commands

Simple explanations for complete beginners

git clone <repository-url>

This command copies (downloads) a remote repository from the internet to your computer. Use it when you want to start working on a project that already exists online.

git add .

or

git add <file>

This command tells Git to start tracking changes to a file. Use it after you make changes and want to include those changes in your next snapshot (commit).

git commit -m "your message"

This command saves your changes in Git with a short message describing what you did. Think of it as taking a snapshot of your project at this moment.

git push

This command uploads your saved changes (commits) from your computer to the remote repository (like GitHub). Use it to share your work with others or back it up online.

git pull

This command downloads the latest changes from the remote repository to your computer. Use it to make sure you have the most up-to-date version of the project.

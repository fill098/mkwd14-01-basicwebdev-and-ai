# Personal Website Example

This is an example folder structure for a small personal website.

## Folder Structure

```
Copilot_example/
├── index.html          # Main webpage file
├── images/             # Image assets folder
│   ├── profile.jpg     # Profile photo
│   ├── travel1.jpg     # Travel photo 1
│   ├── travel2.jpg     # Travel photo 2
│   ├── hobby1.jpg      # Hobby photo 1
│   └── hobby2.jpg      # Hobby photo 2
└── README.md           # This file
```

## Features

The `index.html` file includes:
- Responsive design that works on mobile and desktop
- Profile section with photo
- About me section
- Interests list
- Photo gallery grid
- Contact information
- Clean, modern styling with CSS

## Usage

1. Replace the placeholder image files with your actual photos
2. Update the content in `index.html` with your personal information
3. Open `index.html` in a web browser to view your website
4. Host the files on a web server to make them accessible online

## Image Recommendations

- **Profile photo**: 400x400 pixels, square format
- **Gallery photos**: 800x600 pixels or similar landscape format
- **File formats**: JPG for photos, PNG for graphics with transparency
- **File sizes**: Keep under 500KB for fast loading

## Customization

You can customize the website by:
- Modifying colors in the CSS styles
- Adding more sections
- Including additional images
- Adding more pages and linking them together
- Integrating with CSS frameworks like Bootstrap for more advanced features
